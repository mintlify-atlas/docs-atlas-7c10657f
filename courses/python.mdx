The **Programación en Python** course is EducaPerú's upcoming program for anyone who wants to start programming from scratch and reach a professional level. Python is one of the world's most in-demand languages — used across web development, data science, automation, and artificial intelligence. This course will take you step by step through the language's core syntax, data structures, and functions, then apply that knowledge to real-world projects that you can showcase to future employers.

<Warning>
  This course is not yet available for enrollment. Registration is not open at this time. You can visit the [course catalog](/courses/catalog) to stay updated on the launch date and express your interest.
</Warning>

## What to Expect

<CardGroup cols={2}>
  <Card title="Python Syntax & Fundamentals" icon="code">
    Start from your very first line of code and build a solid grasp of Python's clean, readable syntax. Cover variables, data types, operators, conditionals, and loops through practical exercises.
  </Card>
  <Card title="Data Structures" icon="list">
    Work with Python's built-in structures — lists, tuples, dictionaries, and sets — to store, organize, and manipulate data efficiently in real programming scenarios.
  </Card>
  <Card title="Functions & Modules" icon="puzzle-piece">
    Write reusable, well-structured code using functions, scope, and arguments. Learn how to organize larger programs into modules and leverage Python's extensive standard library.
  </Card>
  <Card title="Real-World Projects" icon="rocket">
    Apply everything you have learned to hands-on projects that simulate professional tasks. Graduate with working code samples and the confidence to tackle new programming challenges independently.
  </Card>
</CardGroup>
