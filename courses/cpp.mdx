The **Programación en C++** course is EducaPerú's upcoming deep-dive into systems programming. Starting from the absolute basics of the C++ language, you will progress through data structures, memory management, and object-oriented design — building the low-level foundation demanded by software engineering, competitive programming, and embedded systems roles. The course is designed for beginners and intermediate learners who want a rigorous, structured path into one of the world's most powerful programming languages.

<Warning>
  This course is not yet available for enrollment. Registration is not open at this time. You can visit the [course catalog](/courses/catalog) to stay updated on the launch date and express your interest.
</Warning>

## What to Expect

<CardGroup cols={2}>
  <Card title="Systems Programming Fundamentals" icon="microchip">
    Understand how C++ interacts with hardware and the operating system. Learn compilation, linking, and the core syntax that underpins high-performance software.
  </Card>
  <Card title="Data Structures" icon="diagram-project">
    Implement and apply arrays, linked lists, stacks, queues, trees, and graphs. Develop the algorithmic thinking needed to choose and use the right structure for any problem.
  </Card>
  <Card title="Memory Management" icon="memory">
    Master pointers, dynamic allocation, stack vs. heap memory, and best practices for avoiding leaks and undefined behavior in production C++ code.
  </Card>
  <Card title="Object-Oriented Design" icon="cube">
    Build modular, maintainable programs using classes, inheritance, polymorphism, and encapsulation. Apply OOP principles to real-world software design challenges.
  </Card>
</CardGroup>
